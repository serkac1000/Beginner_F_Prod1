MIT App Inventor Firebase Authentication Project

Implementation Guides

We've prepared detailed implementation guides to help you build a MIT App Inventor app with Firebase Authentication:

  • Basic Implementation Guide (Plain Text)
  • Detailed Implementation Guide (HTML)
  • Step-by-Step Project Instructions (HTML)

Important: To use these guides, you'll need:

  1. A Firebase project with Authentication enabled
  2. Your Firebase API Key, Project ID, and App ID
  3. MIT App Inventor account (https://ai2.appinventor.mit.edu/)

Getting Started

Follow these steps to implement Firebase Authentication in your MIT App Inventor project:

  1. Review the detailed implementation guides
  2. Open MIT App Inventor and create a new project
  3. Follow the step-by-step instructions to design the UI and add the logic blocks
  4. Test your app with a Firebase test user
